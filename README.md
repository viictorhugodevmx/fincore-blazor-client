# FinCore Blazor Client

Proyecto del programa **Bankaool .NET Fintech Bridge Lab**.

Frontend construido con **Blazor** para practicar componentes, estado, eventos y UI bancaria usando C#.

## Stack

- .NET 8
- Blazor
- C#
- HTML
- CSS

## Analogía JS/TS

| React / Angular | Blazor |
|---|---|
| Component | `.razor` component |
| Props | `[Parameter]` |
| State | propiedades C# del componente |
| Event handler | `@onclick`, `@onchange` |
| Conditional render | `@if` |
| List render | `@foreach` |
| Service injection | `@inject` |

## Comandos

Restaurar:

```bash
dotnet restore

Compilar:

dotnet build

Ejecutar:

dotnet run

Modo watch:

dotnet watch run
